# Frontend

This directory contains the frontend code for the IOCL Refinery Internship project.

## Description

The frontend provides an interactive dashboard and user interface for:
- Visualizing energy and production data
- Displaying efficiency KPIs (Specific Energy Consumption)
- Anomaly detection visualization
- Energy consumption predictions
- AI chatbot for decision support
