# AI/ML

This directory contains the AI and Machine Learning code for the IOCL Refinery Internship project.

## Description

The AI/ML module provides:
- Anomaly detection algorithms for energy consumption patterns
- Predictive models for future energy consumption
- Optimization recommendation engine
- SEC (Specific Energy Consumption) calculations
- AI chatbot capabilities for decision support
